Color Sets

All design elements contain four important colors that form their core styling: The background color, The text color, The link text color, The link mouse hover color. This color combination is defined as a 'color set'. When a color set is applied to a layout element ( i.e. a row or a column ), this element inherits the background, text and link text color of its color set configuration.
